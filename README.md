ZenWear – Pilates Wear Store
---------------------------------
ZenWear is a simple e-commerce web project designed to showcase premium pilates wear. Built with HTML and CSS, this project highlights a curated selection of activewear tailored for comfort, performance, and style during pilates sessions.

As a Reformer Pilates Instructor, this platform is not just a store; it's a personal recommendation hub where I connect my clients (and fellow fitness lovers) with high-quality, trusted pilates wear essentials.

Project Overview
----------------------

ZenWear is a front-end web application that features:

- A clean and minimal user interface
- A responsive layout for different screen sizes
- A product showcase of 6 curated pilates wear items
- Navigation links for easy browsing
- A simple shopping experience design 

Featured Products
-----------------------------------------
The store includes the following six items:
1. Zen Grip Socks – Non-slip socks for stability on the reformer
2. Zen Sports Bra – Supportive and comfortable for workouts
3. Zen Two-piece Set – Stylish activewear with leggings and a sweater top
4. Zen Leggings – High-waisted and breathable
5. Zen Shorts – Lightweight and flexible
6. Zen Hoodie – Comfortable and perfect for warm-ups

Each product includes:

- Image
- Price (in KSH)
- Description
- Rating
- Add-to-cart button (UI only)
